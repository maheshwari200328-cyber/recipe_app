# Recipe App API

This is a simple **RESTful API** built using **Node.js**, **Express**, and **MongoDB** for managing recipes.  
You can **create**, **read**, **update**, and **delete** recipes (CRUD operations).

---

##  Features

- Create a new recipe  
- Get all recipes  
- Get a single recipe by ID  
- Update a recipe by ID  
- Delete a recipe by ID  

---

## Tech Stack

- **Backend Framework:** Node.js + Express  
- **Database:** MongoDB (Local)  
- **Driver:** MongoDB Native Driver
